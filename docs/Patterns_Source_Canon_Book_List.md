# Patterns Closed-Loop Source Canon

This is the public book list for the closed-loop source system used to build the Patterns app brain.

Core reasoning is limited to these books plus reviewed case-study evidence unless a new source is deliberately reviewed and added later.

Case studies are treated as a separate applied reasoning and outcome layer. They help with practitioner-style reasoning, follow-up questions, outcomes, limitations, safety notes, and confidence calibration. They do not replace the core books, repertories, materia medica, or classical foundations.
