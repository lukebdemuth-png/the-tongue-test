# Brand Images

Formal app name: `Empirical Patterns`

Short app name: `Patterns`

Tagline: `3 traditions, one pattern.`

Current logo:

- `patterns-logo-mark.png`
- `3-patterns-logo.png`

Replacement rule:

Keep the same `patterns-logo-mark.png` filename to replace the displayed logo without changing code. If you use a new filename, update `siteConfig.logo` in `lib/site.ts`.

Notes:

- The header uses `patterns-logo-mark.png` as a small square mark.
- `3-patterns-logo.png` is the original full orange image.
- The landing page also uses this file near the hero headline.
- Use a clean PNG or JPG with enough center padding so the mark reads at small sizes.
