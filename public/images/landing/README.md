# Landing Page Images

These images are replaceable placeholders for the 3-patterns landing page.

Current files:

- `hero-patterns-still-life.png` - first viewport hero background.
- `source-library-workflow.png` - source-backed trust section.
- `free-content-newsletter.png` - free content and newsletter section.

Replacement rule:

Keep the same filenames to swap images without changing code. If you want new filenames, update the `landingImages` object in `app/page.tsx`.

Recommended format:

- Wide editorial images work best.
- Avoid readable text inside the image.
- Avoid medical diagnosis, prescription, or treatment imagery.
- Keep the subject honest to the product: source library, practitioner research, careful pattern reasoning, free educational content.
