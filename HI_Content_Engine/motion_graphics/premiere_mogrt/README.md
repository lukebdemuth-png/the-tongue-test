# Himalayan Institute Premiere MOGRT Kit

This folder contains actual `.mogrt` files for Adobe Premiere Pro in:

- [`exports`](./exports)

Use these in Premiere's `Essential Graphics` panel for vertical Instagram Reels.

Templates included:

- `HI__HOOK_TITLE__CENTER.mogrt`
- `HI__QUOTE_OVERLAY__CENTER.mogrt`
- `HI__QUOTE_OVERLAY__LOWER_THIRD.mogrt`
- `HI__KEY_POINT__01.mogrt`
- `HI__KEY_POINT__02.mogrt`
- `HI__KEY_POINT__03.mogrt`
- `HI__SPEAKER_LOWER_THIRD.mogrt`
- `HI__BODY_HIGHLIGHT__CALLOUT.mogrt`
- `HI__BODY_HIGHLIGHT__NUMBERED.mogrt`
- `HI__BODY_HIGHLIGHT__CENTER_TAG.mogrt`
- `HI__CLOSING_CTA__CENTER.mogrt`

Important:

- These are Premiere-ready MOGRT files, not After Effects projects.
- The `3 key points` section is implemented as three separate numbered MOGRTs so it stays easy to reuse and edit in Premiere.
