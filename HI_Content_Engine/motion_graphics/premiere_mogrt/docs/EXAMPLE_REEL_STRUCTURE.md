# Example Reel Structure

## 30-45 Second Reel

1. Hook title for the first `1-2` seconds.
2. Main teaching or practice footage.
3. Add one center quote or lower-third quote during the strongest spoken idea.
4. Add `HI__KEY_POINT__01`, `HI__KEY_POINT__02`, and `HI__KEY_POINT__03` as short staggered takeaways.
5. Add a body-part cue with `HI__BODY_HIGHLIGHT__CALLOUT`, `HI__BODY_HIGHLIGHT__NUMBERED`, or `HI__BODY_HIGHLIGHT__CENTER_TAG` during the pose demonstration.
6. Add `HI__SPEAKER_LOWER_THIRD` when the teacher first appears clearly.
7. End with `HI__CLOSING_CTA__CENTER`.

## Example Timeline

- `00:00-00:02` Hook
- `00:02-00:12` Main clip
- `00:08-00:12` Quote overlay
- `00:13-00:16` Key point 01
- `00:16-00:19` Key point 02
- `00:19-00:22` Key point 03
- `00:22-00:25` Body-part highlight
- `00:25-00:28` Main clip continues
- `00:28-00:30` CTA
