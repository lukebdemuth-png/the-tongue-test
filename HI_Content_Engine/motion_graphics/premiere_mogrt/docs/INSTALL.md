# Install In Premiere

## Option 1

In Premiere Pro:

1. Open `Window > Essential Graphics`.
2. Go to the `Browse` tab.
3. Click the `+` button or `Install Motion Graphics template`.
4. Select the `.mogrt` files from:
   `/Users/creative/Documents/New project/HI_Content_Engine/motion_graphics/premiere_mogrt/exports`

## Option 2

Copy the `.mogrt` files into:

`/Users/creative/Library/Application Support/Adobe/Common/Motion Graphics Templates`

Then restart Premiere if they do not appear immediately.

## Beginner Use

1. Create a `1080 x 1920` sequence.
2. Drag a MOGRT from `Essential Graphics` onto the timeline above your video.
3. Select the graphic clip.
4. Open the `Edit` tab in `Essential Graphics`.
5. Replace the text fields.
6. Adjust clip duration by dragging the clip edges.

## Recommended Reel Order

1. `HI__HOOK_TITLE__CENTER`
2. Main teacher clip
3. `HI__QUOTE_OVERLAY__CENTER` or `HI__QUOTE_OVERLAY__LOWER_THIRD`
4. `HI__KEY_POINT__01`
5. `HI__KEY_POINT__02`
6. `HI__KEY_POINT__03`
7. `HI__BODY_HIGHLIGHT__CALLOUT` or `HI__BODY_HIGHLIGHT__NUMBERED`
8. `HI__SPEAKER_LOWER_THIRD`
9. `HI__CLOSING_CTA__CENTER`

## For Yoga Body-Part Highlights

Use these when you want to label alignment areas during practice footage:

- `HI__BODY_HIGHLIGHT__CALLOUT`
- `HI__BODY_HIGHLIGHT__NUMBERED`
- `HI__BODY_HIGHLIGHT__CENTER_TAG`

Good examples:

- `HANDS` / `Press evenly through the palms`
- `HIPS` / `Draw the hips back`
- `SPINE` / `Lengthen through the crown`
- `SHOULDERS` / `Relax away from the ears`
- `FEET` / `Ground through all four corners`
