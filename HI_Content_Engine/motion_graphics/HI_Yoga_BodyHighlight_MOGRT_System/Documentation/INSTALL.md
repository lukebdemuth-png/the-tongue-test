# Install

1. Open Adobe Premiere Pro.
2. Go to `Window > Essential Graphics`.
3. Open the `Browse` tab.
4. Click `Install Motion Graphics Template`.
5. Select one or more `.mogrt` files from:
   `/Users/creative/Documents/New project/HI_Content_Engine/motion_graphics/HI_Yoga_BodyHighlight_MOGRT_System/MOGRTS`

If you want them available permanently, copy the `.mogrt` files into your Adobe Motion Graphics Templates folder.

- `HI_BodyGlow_Sweep.mogrt`
- `HI_Glow_Point_Node.mogrt`
- `HI_Callout_Line.mogrt`
- `HI_Caption_Callout.mogrt`
- `HI_Breath_Pulse.mogrt`
