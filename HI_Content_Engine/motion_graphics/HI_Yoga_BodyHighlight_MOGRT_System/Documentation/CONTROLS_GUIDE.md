# Controls Guide

## HI_BodyGlow_Sweep

Use for the soft anatomical area glow.

Editable controls available in Premiere:

- cue text
- overall position
- glow color
- glow intensity
- line visibility
- block size

## HI_Glow_Point_Node

Use for the small amber body point.

Editable controls available in Premiere:

- circle color
- circle size
- circle position
- circle opacity

## HI_Callout_Line

Use for the connector line between the body point and caption.

Editable controls available in Premiere:

- start X / Y
- end X / Y
- line color
- point target color
- global size
- global rotation

## HI_Caption_Callout

Use for the premium body-area caption.

Editable controls available in Premiere:

- body area text
- cue text
- duration in timeline

## HI_Breath_Pulse

Use for inhale / exhale moments.

Editable controls available in Premiere:

- cue text
- circle color
- circle size
- circle position
- circle opacity

## HI_Save_CTA

Use as the final end card.

Editable controls available in Premiere:

- CTA text
- accent text

## HI_Full_Cue_Combo

Use when you want a faster one-clip setup with line + target point + caption.

Editable controls available in Premiere:

- cue text
- line color
- point color
- start X / Y
- end X / Y
