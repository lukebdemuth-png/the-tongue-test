# Style Guide

Visual direction:

- premium
- minimal
- cinematic
- warm amber glow
- subtle yoga anatomy guidance

Brand color:

- orange glow: `#CF6F1A`
- warm white text: `#F6F0E8`
- charcoal support tone: `#181512`

Typography:

- primary: `Gotham-Medium`
- fallback: closest available system sans if Gotham is unavailable

Reel format:

- `1080 x 1920`
- keep captions above lower reel caption area
- keep body highlights soft and non-neon

Recommended timing:

- cue in: `0.3s - 0.6s`
- hold: `3s - 4s`
- dissolve out: `0.3s - 0.8s`

Recommended stack:

1. `HI_Glow_Point_Node`
2. `HI_BodyGlow_Sweep`
3. `HI_Callout_Line`
4. `HI_Caption_Callout`

Fast shortcut:

- `HI_Full_Cue_Combo`
