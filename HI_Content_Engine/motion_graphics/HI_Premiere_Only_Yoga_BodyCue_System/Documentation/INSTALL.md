# Install

1. Open Premiere Pro.
2. Create a `1080 x 1920` vertical sequence.
3. Import everything from the `Assets` folder.
4. Open `Window > Essential Graphics` for your text layers.
5. Build each cue from the imported SVG assets plus a text layer.

Recommended bins:

- `Footage`
- `Body Cue Assets`
- `Text Styles`
- `Demo Timeline`

This system is not a MOGRT pack.

It is a Premiere-only reusable overlay system made from importable assets and timeline stacking.
