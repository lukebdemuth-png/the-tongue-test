# Duplicate For Other Poses

1. Duplicate your finished cue stack on the timeline.
2. Drag the duplicated stack to the next time range.
3. Move the glow ring and point dot over the new body area.
4. Rotate or reposition the connector line.
5. Replace the text.
6. Repeat for each cue.

Body areas this package is designed for:

- palms
- wrists
- shoulders
- spine
- hips
- hamstrings
- knees
- calves
- heels
- neck
- jaw
