# Beginner Quickstart

If you do not know After Effects, use this file:

- [downward_dog_one_click_setup.jsx](/Users/creative/Documents/New%20project/HI_Content_Engine/motion_graphics/after_effects/downward_dog_one_click_setup.jsx)

## Exactly what to do

1. Open Adobe After Effects.
2. Go to `File > Scripts > Run Script File...`
3. Choose `downward_dog_one_click_setup.jsx`
4. If the default video path is still available, the script uses it automatically.
5. If not, pick your yoga video clip when the `Open` window appears.
6. Wait for the script to finish building the reel.
7. Look for the comp named `DOWNWARD_DOG__MASTER`
8. Open the `Render Queue` panel.
9. Click `Render`

## What the script does for you

- creates the full 60-second vertical reel
- imports your chosen video
- can auto-use the default video path in the script
- fits the video into the vertical comp
- adds the reel to the Render Queue
- points the output file to your Desktop

## If something looks slightly off

- Find the small layers ending in `*_CTRL`
- Move the matching control a little
- Preview again

Usually you will only need tiny adjustments.

## If the script opens but no video gets added

- Run the script again
- Choose the clip again when the `Open` window appears
- Make sure you picked a real `.mp4` or `.mov` file
