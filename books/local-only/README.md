# Local-Only Source Files

These files exist on the local machine but are intentionally not committed to GitHub.

## Reason

They are raw PDFs, full OCR text, full cleaned markdown, private references, or sources with rights/access status that must be reviewed before repository publication.

## Ignored Patterns

- `data/raw/*.pdf`
- `data/clean/*.md`
- `data/extracted/*.txt`
- `sources/raw/**/*.pdf`
- `private_sources/`
